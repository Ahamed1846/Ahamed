# Ahamed

Personal portfolio for Ahamed Meyan, built with React, Vite, Tailwind CSS, and Framer Motion.

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run lint
npm run build
```

## Deployment

The project is ready for Netlify. Build output goes to `dist`.

```bash
npm run build
```
